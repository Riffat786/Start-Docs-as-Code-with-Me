# 📚 Module 6 · Docs-as-Code Workflow

Docs-as-Code is more than writing documentation in Markdown—it's a collaborative workflow that applies modern software development practices to documentation. By combining version control, code reviews, automation, and continuous publishing, documentation becomes easier to maintain, review, and deliver.

In this module, you'll learn how documentation moves through its lifecycle and how teams collaborate to produce high-quality documentation using Docs-as-Code principles.

---

## 🎯 Learning Objectives

After completing this module, you will be able to:

- Understand the Docs-as-Code lifecycle.
- Follow a documentation branching strategy.
- Collaborate through documentation reviews.
- Manage documentation versions.
- Prepare documentation for releases.
- Apply Docs-as-Code best practices.
