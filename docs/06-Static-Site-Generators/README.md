# 🌐 Module 7 · Static Site Generators

Static Site Generators (SSGs) transform Markdown files into fast, secure, and searchable documentation websites. They are the backbone of many Docs-as-Code solutions and enable documentation teams to publish professional websites directly from source files.

In this module, you'll compare popular documentation platforms and understand when to choose each one for your projects.

---

## 🎯 Learning Objectives

After completing this module, you will be able to:

- Understand how static site generators work.
- Compare popular documentation platforms.
- Select the right tool for different documentation projects.
- Understand the strengths and limitations of each platform.
