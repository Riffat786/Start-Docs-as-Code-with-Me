# 💻 Module 2 · Environment Setup

A well-configured environment is the foundation of an efficient Docs-as-Code workflow.

In this module, you'll install and verify all the tools used throughout the course. By the end, you'll have a complete professional documentation environment ready for Git, GitHub, Docusaurus, AI-assisted writing, and automation.

---

## 🎯 Learning Objectives

After completing this module, you will be able to:

- Install Git.
- Install GitHub Desktop.
- Install Node.js.
- Install Python.
- Install Docker.
- Install Cursor IDE.
- Install Claude Code.
- Verify your development environment.

---

## 📚 Lessons

| Lesson | Description |
|---------|-------------|
| Install Git | Version control system. |
| Install GitHub | Connect to GitHub repositories. |
| Install Node.js | Required for Docusaurus. |
| Install Python | Required for automation. |
| Install Docker | Containerized development. |
| Install Cursor | AI-powered IDE. |
| Install Claude Code | AI coding assistant. |
| Verify Installation | Confirm everything works correctly. |
